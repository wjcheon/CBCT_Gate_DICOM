## About this code  
This code is Conebeam-CT on Gate (Monte-carlo simulation)  

### Geometry  
1) World  
2) Cylinderical phantom (PMMA,TEFLON,AIR)  
3) Silicon detector   

### Source   
1) General particle source(70,80,100,120, and 140 kVp)   

### Scoring    
1) Silicon dectoctor (Voxel geometry 1920 x 1536)
